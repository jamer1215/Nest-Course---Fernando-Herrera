# Ejecutar en desarrollo

1. Clone el repositorio
2. Ejecute:

```
yarn install
```
3. Instale Nest CLI si no lo tiene:
```
npm i -g @nestjs/cli
```

4. Levante la base de datos:
```
docker.compose up -d
```

## Stack usado
* MongoDB
* Nest
